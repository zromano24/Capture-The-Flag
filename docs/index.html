Welcome to this Capture the Flag exercise!

Your task is to put yourself in the shoes of an attacker. There's an election coming up, and your local government has decided to use a cloud-based web application to conduct the election. You think they might have missed some security checks, though, so you're going to read up on common cloud and web app vulnerabilities, then try to penetrate the election website and change the results.

Reading material: OWASP Top 10, Application Security, Common Cloud Misconfigurations
